# 🌾 AgroTech Sustentável

Projeto desenvolvido por Kauan utilizando HTML e CSS com o tema **"Agro Forte: Futuro Sustentável – Equilíbrio entre Produção e Meio Ambiente"**.

## 📌 Objetivo

Apresentar como a tecnologia e a sustentabilidade podem trabalhar juntas para aumentar a produção agrícola e preservar os recursos naturais.

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3


## 📂 Estrutura

```text
└── README.md
```

## 🌱 Tema

Tecnologia, inovação e sustentabilidade no agronegócio.
